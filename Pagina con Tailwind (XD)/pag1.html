Base de datos

¿Que es una base relacional?


-Son las más utilizadas como tecnología para la industria. Están diseñadas para almacenar datos estructurados en tablas relacionadas entre sí. Las tablas están organizadas en filas y columnas y utilizan claves para relacionar los datos entre ellas. Las bases de datos relacionales son muy escalables y ofrecen una gran flexibilidad en cuanto a la forma en que se pueden consultar los datos. es la base de datos mas commercial y de mayor empuje/impacto en el Mercado actual ademas de ser estable y confiable





¿Que es una Base de datos de objetos?


-Son un tipo de base de datos NoSQL que almacenan datos como objetos. Están diseñadas para trabajar con lenguajes de programación orientados a objetos, como Java o Python. crea un orden jerarquico en estos lenguajes que lo ayuda a poder manipular objetos complejos con una presicion infalible





¿Que son las tablas?


-Son la estructura fundamental de una base de datos. Cada tabla almacena datos sobre una entidad específica, como usuarios, productos o transacciones. Las tablas se componen de filas y columnas





¿Que es una llave primaria?


-Una clave primaria es un campo (o combinación de campos) que identifica de manera única cada registro en una tabla. Por ejemplo, el ID de usuario. son de vital importancia para el funcionamiento correcto de la base de datos


¿Que es una base NO-relacional?


-Son una alternativa a las bases de datos relacionales. No utilizan tablas y claves como las bases de datos relacionales, sino que utilizan una variedad de estructuras de datos, como documentos, gráficos y pares clave-valor. se caracateriza por ser de estructura muy escalable y de gran optimizacion y rendimiento dando como resultado un mejor almacenamiento de grandes cantidades de datos como redes sociales por ejemplo





¿Que es un gestor de base de datos?


-también conocido como Sistema de Gestión de Bases de Datos (SGBD) o Database Management System (DBMS), es un sistema software que tiene por objetivo servir de interfaz entre la base de datos de una empresa, el usuario y las aplicaciones. Se compone por un lenguaje de definición, manipulación y consulta de datos. Además, permite garantizar su seguridad e integridad.





¿Que son los registros y campos?


-Cada fila en una tabla representa un registro único. Por ejemplo, en una tabla de usuarios, cada fila podría representar un usuario diferente. Las columnas definen los atributos de los datos almacenados en la tabla. Por ejemplo, en una tabla de usuarios, las columnas podrían ser ID, Nombre, Correo Electrónico, etc.





¿Que es una llave candidata?


-Es una superclave mínima. Esto significa que es una combinación de atributos que puede identificar de manera única cada fila en una tabla, y no contiene atributos redundantes, como su nombre indica se utiliza para identificar elementos de la misma tabla que no son de tanta importancia como para los que se require una primary key
